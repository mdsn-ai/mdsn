# Elements 组件

`@mdsnai/sdk/elements` 提供官方默认 Web Components UI。

常见使用：

```ts
import { mountMdsnElements } from "@mdsnai/sdk/elements";
import { createHeadlessHost } from "@mdsnai/sdk/web";
```

适合：

- 想要开箱可用的默认 UI
- 不想自己实现 block/form/action 渲染层
