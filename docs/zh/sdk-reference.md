# SDK 参考

`@mdsnai/sdk` 为单包发布，使用子路径导出。

## 入口

- `@mdsnai/sdk`
- `@mdsnai/sdk/core`
- `@mdsnai/sdk/server`
- `@mdsnai/sdk/web`
- `@mdsnai/sdk/elements`

## 能力映射

- `core`: 解析/校验/序列化/协商
- `server`: 运行时与 Node Host
- `web`: 浏览器 headless 运行时
- `elements`: 官方默认 UI
